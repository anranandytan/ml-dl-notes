    04 — Bayesian Regression

    Keywords: prior, posterior, conjugate prior, completing the square, predictive distribution, epistemic uncertainty, aleatoric uncertainty, marginal likelihood, kernel trick, Gaussian process

    1. Motivation

    Standard (frequentist) linear regression finds a single weight vector \(w\) by minimising training loss. This approach tells you nothing about how confident you should be in that estimate—with 5 data points in a 100-dimensional space, the MLE solution is wildly underdetermined, yet it produces a single point estimate with no attached uncertainty.

    Bayesian regression replaces the single point estimate with a full probability distribution over weights, \(p(w \mid \mathcal{D})\). We begin with a prior \(p(w)\) encoding initial beliefs, observe data, and update to the posterior \(p(w \mid \mathcal{D})\) via Bayes' theorem. Predictions then average over all plausible weight vectors, weighted by their posterior probability.

    Key benefits:

    Benefit What it means
    Uncertainty quantification Predictive error bars widen in data-sparse regions
    Automatic regularisation The prior penalises large weights without a tuned \(\lambda\)
    Bayesian Occam's Razor The marginal likelihood penalises unnecessarily complex models
    Hyperparameter selection Optimise \(\sigma^2\), \(\Sigma_p\) via marginal likelihood rather than cross-validation

    2. Historical Notes

    Year Event
    1763 Thomas Bayes' posthumous essay introduces Bayes' theorem
    1812 Laplace independently develops Bayesian probability
    1970s Conjugate prior analysis systematised; analytic posteriors become textbook material
    1992 MacKay's evidence framework provides principled hyperparameter selection
    1996–98 Rasmussen & Williams reinterpret kernelised Bayesian regression as Gaussian Process Regression

    3. MAP Estimation: Ridge and Lasso as Special Cases

    Before the full Bayesian treatment, consider Maximum A Posteriori (MAP) estimation, which finds the mode of the posterior rather than the full distribution.

    By Bayes' theorem:

    \[p(w \mid \mathcal{D}) \propto p(Y \mid X, w)\, p(w)\]

    MAP maximises this; taking \(\log\) (monotone, so argmax unchanged):

    \[w_{\text{MAP}} = \arg\max_w \bigl[\log p(Y \mid X, w) + \log p(w)\bigr]\]

    3.1 Gaussian Prior → Ridge Regression

    With Gaussian noise (\(\sigma^2\) known) and zero-mean isotropic Gaussian prior \(p(w) = \mathcal{N}(0, \tau^2 I)\):

    \[\log p(Y \mid X, w) = -\frac{1}{2\sigma^2}\|Y - Xw\|^2 + \text{const}, \qquad \log p(w) = -\frac{1}{2\tau^2}\|w\|_2^2 + \text{const}\]

    The MAP objective becomes:

    \[w_{\text{MAP}} = \arg\min_w \left[\|Y - Xw\|^2 + \underbrace{\frac{\sigma^2}{\tau^2}}_{=:\lambda}\|w\|_2^2\right]\]

    This is exactly Ridge regression. A tighter prior (small \(\tau^2\)) means stronger regularisation (large \(\lambda\)).

    3.2 Laplace Prior → Lasso

    With independent Laplace priors \(p(w_j) \propto \exp(-|w_j|/b)\):

    \[w_{\text{MAP}} = \arg\min_w \left[\|Y - Xw\|^2 + \frac{2\sigma^2}{b}\|w\|_1\right]\]

    This is Lasso. The Laplace distribution peaks sharply at zero; the level sets of \(\|\cdot\|_1\) are diamond-shaped with corners on the coordinate axes, so the optimal solution tends to have some coordinates exactly zero (sparsity).

    3.3 Limitation of MAP

    MAP collapses the posterior to a point, discarding all uncertainty. It tells you where the posterior peaks but not how wide it is.

    4. Bayesian Linear Regression: Setup

    Data. \(\mathcal{D} = \{(x_i, y_i)\}_{i=1}^N\) with \(x_i \in \mathbb{R}^p\), \(y_i \in \mathbb{R}\). In matrix form:

    \[X \in \mathbb{R}^{N \times p}, \qquad Y \in \mathbb{R}^N\]

    Likelihood. Observations are generated by a linear model with i.i.d. Gaussian noise:

    \[y_i = w^\top x_i + \varepsilon_i, \quad \varepsilon_i \overset{\text{iid}}{\sim} \mathcal{N}(0, \sigma^2) \qquad\Longleftrightarrow\qquad Y \mid X, w \sim \mathcal{N}(Xw, \sigma^2 I)\]

    Note: \(\sigma^2 I\) is the covariance (not precision). The precision is \((\sigma^2 I)^{-1} = \sigma^{-2}I\).

    Prior. Zero-mean Gaussian over the weights:

    \[p(w) = \mathcal{N}(0, \Sigma_p)\]

    Large \(\Sigma_p\) = weak regularisation; small \(\Sigma_p\) = strong regularisation.

    5. Posterior Derivation

    5.1 Why the Posterior Is Gaussian (Conjugacy)

    The likelihood \(p(Y \mid X, w)\) is Gaussian in \(w\) (the exponent is quadratic in \(w\)). The prior \(p(w)\) is also Gaussian in \(w\). Their product is the exponential of a sum of two quadratics in \(w\), which is the exponential of a quadratic—still a Gaussian. This self-conjugacy of the Gaussian family means the posterior has a closed form.

    5.2 Log-Posterior

    \[\log p(w \mid \mathcal{D}) \propto \log p(Y \mid X, w) + \log p(w)\]
    \[= -\frac{1}{2\sigma^2}(Y - Xw)^\top(Y - Xw) - \frac{1}{2}w^\top \Sigma_p^{-1} w\]

    Expanding \((Y - Xw)^\top(Y - Xw)\). Note that for any vector \(v\), \(v^\top v = \|v\|^2\), and a scalar equals its own transpose:

    \[(Y - Xw)^\top(Y - Xw) = Y^\top Y - 2\,w^\top X^\top Y + w^\top X^\top X w\]

    (The two cross-terms \(Y^\top Xw\) and \((Xw)^\top Y = w^\top X^\top Y\) are equal scalars, so they combine with factor 2.)

    Dropping the constant \(Y^\top Y\) and grouping:

    \[\log p(w \mid \mathcal{D}) \propto \frac{1}{\sigma^2}\,w^\top X^\top Y - \frac{1}{2}\,w^\top\underbrace{\left(\frac{1}{\sigma^2}X^\top X + \Sigma_p^{-1}\right)}_{:=\,A}w\]

    5.3 Completing the Square

    A Gaussian \(\mathcal{N}(\mu_w, A^{-1})\) has log-kernel \(-\tfrac{1}{2}(w - \mu_w)^\top A(w - \mu_w) = -\tfrac{1}{2}w^\top A w + w^\top A\mu_w + \text{const}\).

    Matching the linear-in-\(w\) term:

    \[A\mu_w = \frac{1}{\sigma^2}X^\top Y \qquad\Longrightarrow\qquad \mu_w = \frac{1}{\sigma^2}A^{-1}X^\top Y\]

    5.4 Result

    \[\boxed{p(w \mid \mathcal{D}) = \mathcal{N}(\mu_w, \Sigma_w)}\]
    \[A = \frac{1}{\sigma^2}X^\top X + \Sigma_p^{-1}, \qquad \Sigma_w = A^{-1}, \qquad \mu_w = \frac{1}{\sigma^2}A^{-1}X^\top Y\]

    Interpretation: - \(A\) is the posterior precision. It equals the sum of data precision \(\frac{1}{\sigma^2}X^\top X\) and prior precision \(\Sigma_p^{-1}\). Information adds in precision space. - As \(N \to \infty\): \(\mu_w \to w_{\text{OLS}}\), \(\Sigma_w \to 0\) (posterior concentrates on OLS solution). - As \(N \to 0\): \(\mu_w \to 0\), \(\Sigma_w \to \Sigma_p\) (posterior reverts to prior).

    6. Predictive Distribution

    6.1 Latent Function Value

    For a new input \(x^{\ast} \in \mathbb{R}^p\), the latent function value \(f(x^{\ast}) = w^\top x^{\ast}\) is a linear function of \(w\). Using the standard result for linear transformations of Gaussians (\(c^\top w \sim \mathcal{N}(c^\top \mu_w, c^\top \Sigma_w c)\) when \(w \sim \mathcal{N}(\mu_w, \Sigma_w)\)):

    \[f(x^{\ast}) \sim \mathcal{N}\bigl((x^{\ast})^{\top} \mu_w, (x^{\ast})^{\top} \Sigma_w\, x^{\ast}\bigr)\]

    6.2 Observed Output

    The observed \(y^{\ast} = f(x^{\ast}) + \varepsilon^{\ast}\) adds independent noise \(\varepsilon^{\ast} \sim \mathcal{N}(0, \sigma^2)\). The variance of the sum of two independent random variables is the sum of their variances:

    \[\boxed{p(y^{\ast} \mid \mathcal{D}, x^{\ast}) = \mathcal{N}\Bigl((x^{\ast})^{\top} \mu_w, \underbrace{(x^{\ast})^{\top} \Sigma_w\, x^{\ast}}_{\text{epistemic}} + \underbrace{\sigma^2}_{\text{aleatoric}}\Bigr)}\]

    6.3 Two Types of Uncertainty

    Type Source Reducible with more data?
    Epistemic \((x^{\ast})^{\top} \Sigma_w x^{\ast}\) Uncertainty about the weights Yes — \(\Sigma_w \to 0\) as \(N \to \infty\)
    Aleatoric \(\sigma^2\) Irreducible observation noise No

    Epistemic uncertainty depends on \(x^{\ast}\): it is large where training data is sparse and small near training points.

    7. Marginal Likelihood and Hyperparameter Selection

    To select hyperparameters \(\theta = (\sigma^2, \Sigma_p)\) without cross-validation, maximise the marginal likelihood (also called the evidence):

    \[p(Y \mid X, \theta) = \int p(Y \mid X, w, \sigma^2)\, p(w \mid \Sigma_p)\, dw\]

    Both factors are Gaussian in \(w\), so the integral is analytic (via completing the square in \(w\)). The result is:

    \[p(Y \mid X, \theta) = \mathcal{N}(Y \mid 0, \sigma^2 I + X \Sigma_p X^\top)\]

    The log marginal likelihood decomposes as:

    \[\log p(Y \mid X) = -\frac{1}{2}Y^\top C^{-1}Y - \frac{1}{2}\log|C| - \frac{N}{2}\log(2\pi), \qquad C = \sigma^2 I + X\Sigma_p X^\top\]
    • Term 1 (\(-\frac{1}{2}Y^\top C^{-1}Y\)): data fit — larger when the model explains \(Y\) well.
    • Term 2 (\(-\frac{1}{2}\log|C|\)): complexity penalty — larger \(\Sigma_p\) or smaller \(\sigma^2\) increases \(|C|\), penalising flexible models (Bayesian Occam's Razor).

    Optimise over \(\theta\) using gradient ascent. This is Empirical Bayes or Type-II Maximum Likelihood (MacKay, 1992).

    8. Kernelised Bayesian Regression and Gaussian Processes

    8.1 Feature Maps

    Replace the linear model \(f(x) = w^\top x\) with \(f(x) = w^\top \phi(x)\), where \(\phi: \mathbb{R}^p \to \mathcal{H}\) is a (possibly infinite-dimensional) feature map. All derivations above hold with \(X\) replaced by the feature matrix \(\Phi\).

    8.2 The Kernel Trick

    The posterior mean and predictive formulas depend on features only through pairwise inner products:

    \[k(x, x') := \phi(x)^\top \Sigma_p\, \phi(x')\]

    This is the kernel function. Define the \(N \times N\) Gram matrix \(K\) with \(K_{ij} = k(x_i, x_j)\) and the \(N\)-vector \(k^{\ast}\) with \(k^{\ast}_i = k(x_i, x^{\ast})\). The predictive distribution becomes:

    \[p(y^{\ast} \mid \mathcal{D}, x^{\ast}) = \mathcal{N}\Bigl((k^{\ast})^{\top}(K + \sigma^2 I)^{-1}Y, k(x^{\ast}, x^{\ast}) - (k^{\ast})^{\top}(K + \sigma^2 I)^{-1}k^{\ast} + \sigma^2\Bigr)\]

    This is the Gaussian Process Regression formula. The weight-space and function-space views are mathematically equivalent (connected via the Woodbury identity); the GP view avoids working in the infinite-dimensional feature space.

    8.3 Common Kernels

    Kernel Formula Implicit feature space
    Linear \(x^\top x'\) \(\mathbb{R}^p\)
    Polynomial \((x^\top x' + 1)^d\) All monomials up to degree \(d\)
    Squared Exponential (RBF) \(\sigma_f^2\exp(-\|x-x'\|^2 / 2\ell^2)\) Infinite-dimensional

    9. Summary

    Model:
  y_i = wᵀx_i + ε_i,    ε_i ~ N(0, σ²)
  p(w) = N(0, Σ_p)

Posterior (Gaussian × Gaussian = Gaussian):
  A = (1/σ²)XᵀX + Σ_p⁻¹
  p(w|D) = N(μ_w = (1/σ²)A⁻¹XᵀY,   Σ_w = A⁻¹)

Predictive:
  p(y*|D,x*) = N( (x*)ᵀμ_w,   (x*)ᵀΣ_w x* + σ² )
                              ↑ epistemic     ↑ aleatoric

Marginal likelihood:
  p(Y|X,θ) = N(Y | 0, σ²I + XΣ_pXᵀ)    [optimise over θ = (σ², Σ_p)]

Kernel trick:
  k(x,x') = φ(x)ᵀΣ_p φ(x')   →   GP regression formula
